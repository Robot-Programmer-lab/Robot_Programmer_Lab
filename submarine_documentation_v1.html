Разработка подводной лодки АД-1

Цель проекта: Содать подводную лодку на проводном управлении,
Для исследования водоемов.

Задачи:
1. Создать подводную лодку
2. Протестировать подлодку
3. Исправить выявленные недостатки во время тестирования
4. Провести исследование водоема с помошью подводной лодки
